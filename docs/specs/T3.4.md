# T3.4 Legacy Translation Factory

Status: verified locally

## Contract

`forge-porttools translate --all --jobs N` parses the pinned legacy corpus,
maps complete single-face scripts through T3.3, emits canonical `.frs` files,
and compiler-roundtrips every emitted definition. A card is emitted only when
its catalog identity, printed characteristics, keywords, root abilities, SVar
links, and typed expressions all translate exactly. No partial card is written.

The generated directory is tool-owned and marked before it may be replaced.
Parallel workers perform read/parse/map/compile work; results and quarantine
records are sorted before deterministic sequential output. Duplicate catalog
names, unsupported layouts, parameterized keywords, metadata failures, mapper
gaps, and compiler failures receive stable file-level quarantine codes.

## Local Evidence

The 16-worker campaign considered 33,290 scripts, emitted 6,289 canonical
definitions (18.8916%), and quarantined 27,001 without partial output. A repeat
campaign produced the same complete output-tree SHA-1
`f71aad3d2285d54911afeafa580c5436a445f347`. `forge-cardc` then built and
roundtripped all 6,289/6,289 definitions against 38,306 catalog identities and
113,234 printings. Local tests, clippy with warnings denied, 235 recurring
oracles, and the 80% coverage gate pass without network or GitHub Actions.
