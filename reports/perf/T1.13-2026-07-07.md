## Baseline Metrics

Values are Criterion mean point estimates in nanoseconds for each benchmarked
iteration.

- `kernel_clone_200_card_state_x64`: 28,434.496 ns
- `kernel_legal_actions_priority_x128`: 6,027.067 ns
- `kernel_apply_pass_priority_x64`: 6,162.291 ns
- `kernel_full_playout_four_turns`: 879,007.171 ns

The batched clone baseline implies roughly 444 ns per 200-card state clone on
this local Mac. That is above the master plan's Section 1.2 target of 200 ns, so
clone performance remains a performance-risk item even though the relative
regression gate is now in place.
